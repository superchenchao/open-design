Implementation · Design Memory

设计记忆 怎么沉淀

每次选择都沉淀为 Design System、偏好和记忆，下次生成更接近你要的结果 —— 越用越懂你。

# skills/design-memory/SKILL.md
name: design-memory
trigger: "after every generation"

on_pick:    record(palette, type, spacing)
on_edit:    record(diff, reason="user-tweak")
on_ship:    record(artifact, design_system)

next_run:
  - load preferences → seed generation
  - apply design-system tokens
  - keep artifacts local in .od/, git-diffable
