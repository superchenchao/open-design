从想法到交付，5 步完成。

Agent 读取全部上下文，产出真实可运行的文件，沙盒内即时预览和修改。每次选择都成为下次生成的上下文，越用越精准。

workflow · 5 steps

1. 选择起点——一句话描述目标，或从模板 / Plugin 起步
2. 确定视觉方向——色板、字体、间距自动带入
3. 生成 Artifact——真实可运行的文件，沙盒预览
4. 交付或制作视频——工程接手，或转营销短片
5. 记忆沉淀——成为下次生成的上下文
